Detailed Description

These macros check that two expressions are not equal in value.

Note:
The types of the two operands need not be the same, as long as comparing them for inequality is supported, as is illustrated in the following example:
00001 #include "illuminate.hpp"
00002 #include <string>
00003 
00004 TEST_CASE(NE_using_strings) {
00005     std::string hello_world = "Hello, world!";
00006     EXPECT_NE_VAL(hello_world,"Hello, world!")
00007     EXPECT_NE_VAL(hello_world,"Hello, planet!")
00008 }
