Detailed Description

Result visitor that prints all tests and their test results.

Note:
Users of this class need to subclass it and supply an implementation of ResultVisitor::test that calls the appropriate test___() methods of this class; alternatively one can mix in either FutureResultVisitor or RunnerResultVisitor.
